  • Applied Quantum Computing - CSCI 420

    Course Description

    • This course introduces the practice of quantum computation and algorithms, in computer science. Topics covered include the mathematics to understand the relationship between physical systems and quantum computation. Algorithms to be implemented include Deutsch's mapped function algorithm, Grover's search algorithm, Simon's periodicity correction, and quantum cryptogrophy. Students will develope their code on a quantum computer.
    • Understand relevant mathematical concepts pertaining to quantum computing for computer science.
    • Be familiar with the milestones and recent developments in quantum computing for computer science.
    • Implement algorithms used in quantum computing that include Deutsch's mapped function algorithm, Grover's search algorithm, Shor's prime factorization algorithm, and Simon's algorithm.
    • Develop, execute, and interpret the results of high-level circut programs for the IBM Q quantum machine.
    • Discuss and interpret quantum computing research artiles pertaining to the computer science field.
    • Learn more about the current state-of-the-art in quantum computing research and where to find more information and related oppertunities.

    Skills Aquired

    • Quantum Computation Algorithms
    • Mathematics for Quantum Computation
    • Designing of Quantum Circuts for IBM Q

  • Computer Networks - CS 441

    • Computer network analysis and design. Fundamentals of data communications: media, transmission, encoding, error detection and handling, link control, packet switching. Network architecture and topology. Network, Transport, and Application layer protocols, services, design issues and performance. Programming assignments using TCP/IP.
    • Compare and contrast techniques for transmitting data over wired and wireless media;
    • Evaluate the performance of reliable link protocols using sliding window protocols;
    • Analyze the merits of different LAN and WAN topologies
    • Write software to implement a networked application using the socket API;
    • Evaluate methods for providing quality of service in networks.

    • Analyze the complexity of computational problems in different domains.
